Use this tool when you have completed the lore planning phase and are ready to
exit the lore planning agent.

This tool will ask the user if they want to switch to the build agent to start
writing the character card from the plan.

Call this tool:
- After you have written a complete character plan to the plan file
- After you have clarified any open questions with the user
- When you are confident the plan is ready to build from

Do NOT call this tool:
- Before you have created or finalized the plan
- If you still have unanswered questions about the character
- If the user has indicated they want to keep planning
